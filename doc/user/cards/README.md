# LCARdS Card Types

Complete reference for all LCARdS card types.

## Card Types

### 🔘 [Button Card](button.md)
SVG-based button card with multiple presets (lozenge, bullet, capped, Picard variants).

**Use cases**: Action buttons, entity state display, custom controls

### 🔲 [Elbow Card](elbow.md)
LCARS elbow/corner designs with segmented displays.

**Use cases**: Dashboard corners, section headers, status indicators

### 🎚️ [Slider Card](slider.md)
Interactive sliders and multimeter gauges.

**Use cases**: Dimmers, volume controls, temperature settings, visual gauges

### 📊 [Chart Card](chart.md)
ApexCharts integration with LCARS theming.

**Use cases**: Historical data visualization, sensor trends, energy monitoring

### 📋 [Data Grid Card](data-grid.md)
Tabular data display with cascading animations.

**Use cases**: Status grids, entity lists, multi-sensor dashboards
