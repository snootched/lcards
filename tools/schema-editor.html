    LCARdS Schema Editor Help
    
            
    
                
    Quick Start
    
                
    1. Click "Add Property" to create a new schema property
    
                
    2. Select a property type and configure constraints
    
                
    3. Add x-ui-hints for visual editor support
    
                
    4. Use templates for common patterns
    
                
                
    Keyboard Shortcuts
    
                
    Ctrl/Cmd + S - Save schema
    
                
    Ctrl/Cmd + Z - Undo
    
                
    Ctrl/Cmd + Shift + Z - Redo
    
                
    Delete - Delete selected property
    
                
    Ctrl/Cmd + D - Duplicate property
    
                
                
    Choose Selector Wizard
    
                
    For oneOf schemas, use the choose selector to provide a visual type picker:
    
                
    1. Create oneOf branches with titles
    
                
    2. Select "choose" selector in x-ui-hints
    
                
    3. Click "Auto-Generate Choices"
    
                
                
    Home Assistant Selectors
    
                
    Available selector types: text, number, boolean, entity, select, icon, ui_color, choose, object, duration, time, date, area, device, addon
